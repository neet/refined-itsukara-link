# Refined Itsukara.link とは

この web サイトはバーチャルユーチューバー事務所「にじさんじ」が提供する公式スケジューラー「いつから.link」をファンが非公式にリデザインしたものです。このサイトは

- いちから株式会社
- にじさんじ

とは一切の関係がありません。
